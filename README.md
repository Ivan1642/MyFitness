# MyFitness
MyFitness
Proyecto Intermodular 2ºDAW Curso 2025/2026

My Fitness es una aplicación web que tiene como finalidad que el usuario pueda registrar sus entrenamientos, guardar ejercicios o crear los suyos propios, a la vez que va observando su progresión de un entrenamiento a otro. La idea es que la web sea como una red social, como Instagram, pero para los entrenamientos, se podrá seguir a otros usuarios para ver sus entrenamientos o guardar sus rutinas de entrenamiento, así como dar me gusta a los entrenamientos. Habrá dos feeds a los que acceder cada usuario, un feed donde aparecerá únicamente la gente que sigues y sugerencias de seguimiento, y otro en el que aparecerán usuarios que no conoces, este seria el feed global. Tambien, cada usuario podrá registrar las medidas de cada parte de su cuerpo, su altura y su peso en el perfil. Esta web será totalmente responsive, porque la idea es que se pueda utilizar desde el teléfono para registrar los entrenamientos en el gimnasio.

Mirando al futuro, la idea es añadir estadísticas para cada usuario, estas estadísticas consistirían en las repeticiones, el peso y la duración de los ejercicios en una parte del cuerpo (ejemplo: pecho, brazo, pierna, espalda) y también la posibilidad de comparar tus estadísticas con las de los usuarios que se tenga agregados (amigos).
